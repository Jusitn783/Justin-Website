Game Development

In my current game project, the main character is struggling with schizophrenia and processing grief after a tragic event. Here’s a breakdown of how I approach creating this game:

  • Concept & Storyline: I start with a clear idea of what the story is about and how it will play out. The story for this game is driven by emotions and mental health struggles, which will be reflected in the gameplay.
  • Game Design: I then create a rough design of how the game will look, what mechanics will be included, and how the user will interact with it.
  • Programming: I use JavaScript for the game logic, focusing on managing game states (e.g., intro, main game, end). The character's behavior, including interactions with the environment, is handled through conditionals and event listeners.
  • Debugging: I constantly test the game to find and fix issues. This could involve fixing problems with the character’s movement or ensuring the UI is functioning correctly.

Website Development

For my personal website, I focus on making sure the design is clean, easy to navigate, and showcases my work. Here’s how I go about it:

  • Layout & Structure: I use HTML to structure the content, ensuring it’s easy to read and responsive. I also make use of CSS to create a layout that adjusts based on the screen size.
  • Design Choices: I choose a minimalistic style with dark colors to make the content stand out and reduce distractions for the user.
  • Functionality: I use JavaScript to add interactivity to the website, like animations and transitions when the user clicks or scrolls.
  • Testing & Optimization: I frequently test my site on different devices to make sure it functions as expected and adjust the code where necessary.
